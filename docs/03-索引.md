## 横向依赖总表

| 来源 | 依赖 | 说明 |
|---|---|---|
| M01-Tabbit协议客户端 | M07-配置密钥 | 读取 cookie/session、脱敏日志配置 |
| M02-账号池调度 | M01-Tabbit协议客户端 | 用协议客户端验证账号可用性 |
| M02-账号池调度 | M05-权益额度维护 | 根据额度状态决定账号是否可用 |
| M03-YYDS邮箱接码 | M07-配置密钥 | 读取 `YYDS_MAIL_API_KEY` |
| M04-账号注册初始化 | M03-YYDS邮箱接码 | 获取注册邮箱和验证码 |
| M04-账号注册初始化 | M01-Tabbit协议客户端 | 调用注册/登录相关协议 |
| M04-账号注册初始化 | M05-权益额度维护 | 注册后触发权益初始化 |
| M05-权益额度维护 | M01-Tabbit协议客户端 | 使用账号 session 调用权益/额度接口 |
| M06-兼容网关 | M02-账号池调度 | 每次请求选择账号 |
| M06-兼容网关 | M01-Tabbit协议客户端 | 发送消息并解析响应 |
| M08-观测运维 | M02/M05/M06 | 聚合状态、日志、错误分类 |

## 阅读顺序

1. 先读 [项目说明书](00-项目说明书.md) 和 [需求文档](01-需求文档.md)。
2. 再读 [架构文档](02-架构文档.md) 与本索引。
3. 实现时按 [开发追踪](04-开发追踪.md) 推荐顺序阅读模块文档。
4. 编写代码时同步更新 [数据字典](06-数据字典.md)、[API 文档](07-API文档.md)、[测试用例](08-测试用例.md) 和 [实现接口参考](09-实现接口参考.md)；涉及流式输出时同步更新 [流式 SSE 链路](10-流式SSE链路.md)；涉及 Codex/Claude/三方客户端接入时同步更新 [Codex Claude 与三方工具接入](11-Codex-Claude与三方工具接入.md)；涉及 403/风控时同步更新 [账号风控与 403 排障](12-账号风控与403排障.md)；涉及真实 endpoint、签名或端到端验收时同步更新 [真实协议校准与端到端验收](13-真实协议校准与端到端验收.md)。
5. 实现或扩展 M01 附件上传时先读 [Protocol Attachment Upload Skeleton 计划](plans/2026-07-02-protocol-attachment-upload-skeleton.md)，并确认 `sendMessage({ attachments })` 与上传路径骨架的边界。
6. 实现 HTTP 路由前先读 [HTTP server 实施计划](plans/2026-07-02-http-server-foundation.md)；扩展 OpenAI/Anthropic `stream:true` 行为时先读 [流式 SSE 链路](10-流式SSE链路.md)，再读 [OpenAI SSE fallback 计划](plans/2026-07-02-openai-sse-fallback.md)、[Anthropic SSE fallback 计划](plans/2026-07-02-anthropic-sse-fallback.md)、[HTTP async SSE flush 计划](plans/2026-07-02-http-async-sse-flush.md)、[HTTP async SSE 错误帧计划](plans/2026-07-02-http-async-sse-error-frames.md) 和 [HTTP async SSE 客户端断开计划](plans/2026-07-02-http-async-sse-client-disconnect.md)。
7. 实现或扩展 M04 账号注册初始化时先读 [AccountProvisioner foundation 计划](plans/2026-07-02-account-provisioner-foundation.md)。
8. 实现或扩展 M05 权益维护时先读 [BenefitsMaintainer foundation 计划](plans/2026-07-02-benefits-maintainer-foundation.md)。
9. 实现或扩展 M08 观测运维时先读 [Observability foundation 计划](plans/2026-07-02-observability-foundation.md) 和 [Ops CLI foundation 计划](plans/2026-07-02-ops-cli-foundation.md)、[Accounts Probe CLI foundation 计划](plans/2026-07-02-accounts-probe-cli-foundation.md)、[Protocol Probe Fixture foundation 计划](plans/2026-07-02-protocol-probe-fixture-foundation.md)、[Fixture CLI foundation 计划](plans/2026-07-02-fixtures-cli-foundation.md)、[Protocol Probe Input CLI foundation 计划](plans/2026-07-02-protocol-probe-input-cli-foundation.md)、[Protocol Probe Template CLI foundation 计划](plans/2026-07-02-protocol-probe-template-cli-foundation.md)、[Protocol Probe Input Schema CLI foundation 计划](plans/2026-07-02-protocol-probe-input-schema-cli-foundation.md)、[Protocol Client Verify Session foundation 计划](plans/2026-07-02-protocol-client-verify-session-foundation.md)。
