## 每次实现前检查

1. 找到对应模块文档。
2. 确认输入、输出、状态变更和失败分类。
3. 先补测试或 fixture，再写实现。
4. 修改公开 API、配置或数据结构后，同步更新本目录文档。
5. 修改 src/index.js 导出后，同步更新 [09-实现接口参考](09-实现接口参考.md)。
6. 实现 HTTP server 前，先执行 [HTTP server 实施计划](plans/2026-07-02-http-server-foundation.md) 的 TDD 清单；扩展 OpenAI `stream:true` 时先读 [OpenAI SSE fallback 计划](plans/2026-07-02-openai-sse-fallback.md)，扩展 Anthropic `stream:true` 时先读 [Anthropic SSE fallback 计划](plans/2026-07-02-anthropic-sse-fallback.md)。
