# 创建临时邮箱

## 目标

通过 YYDS Mail 创建可接收 Tabbit 验证码的临时邮箱。

## 请求

~~~text
POST https://maliapi.215.im/v1/accounts
X-API-Key: ${YYDS_MAIL_API_KEY}
Content-Type: application/json
~~~

body 草案：

~~~json
{
  "localPart": "tabbit-abc123",
  "domain": "public.example.com",
  "subdomain": "team-a"
}
~~~

`localPart` 是推荐字段；`domain` 和 `subdomain` 是否必需取决于账号配置和 YYDS Mail 可用域。

## 返回归一化

~~~ts
type CreateInboxResult = {
  id: string;
  address: string;
  tempTokenRef?: string;
  expiresAt?: string;
};
~~~

## 边界条件

- 域名不可用：返回 `mail_domain_unavailable`。
- localPart 冲突：可自动换后缀，最多重试 3 次。
- 429 限流：读取 `Retry-After`，不忙等。
- API key 无效：立即停止注册流程，提示配置错误。

## 测试

- 请求头包含 `X-API-Key`。
- localPart/domain/subdomain body 构造正确。
- `429` 转换为可退避错误。
- 创建成功后不在日志输出完整邮箱和 token。
