# 提取验证码

## 目标

从邮件主题、文本正文、HTML 正文或 raw source 中提取 Tabbit 注册验证码。

## 规则

优先级：

1. 明确包含 “验证码/code/verification” 上下文的 4-8 位数字。
2. 主题中的 4-8 位数字。
3. 正文中唯一的 4-8 位数字。
4. 多个候选时返回 `ambiguous_code`，不随机选择。

## 接口草案

~~~ts
type ExtractCodeInput = {
  subject?: string;
  text?: string;
  html?: string;
  raw?: string;
  minLength?: number;
  maxLength?: number;
};

type ExtractCodeResult = {
  code: string;
  source: "subject" | "text" | "html" | "raw";
};
~~~

## 边界条件

- HTML 需要先转文本，避免标签拆分数字。
- 邮件中可能包含时间、年份、工单号，不能简单取第一个数字。
- 验证码可能有空格分隔，如 `1 2 3 4 5 6`。
- 提取失败时保留脱敏摘要，方便排查模板变更。
